# Ecommerce Frontend – Módulo 2

Este proyecto corresponde al Módulo 2 de la cursada y consiste en el desarrollo del frontend de un e-commerce simple (MVP).

El objetivo es aplicar HTML5 semántico, Bootstrap para el diseño responsivo y JavaScript básico para la interacción, simulando el funcionamiento de una tienda online.

---

## Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap 5 (vía CDN)
- JavaScript
- Git y GitHub

---

## Estructura del proyecto

El proyecto cuenta con las siguientes páginas principales:

- **Home (index.html):** muestra una grilla de productos utilizando cards de Bootstrap.
- **Detalle de producto (producto.html):** muestra información detallada de un producto seleccionado.
- **Carrito (carrito.html):** carrito de compras simulado con listado de productos y total.

Además, incluye archivos separados para estilos y lógica:

- `css/styles.css`
- `js/main.js`

---

## Funcionalidades

- Visualización de productos en formato de cards.
- Navegación entre páginas mediante navbar.
- Botón "Agregar al carrito" que actualiza el contador en la barra de navegación.
- Diseño responsivo adaptable a dispositivos móviles y escritorio.
